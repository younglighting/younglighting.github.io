Python-笔记

2020-07-06
  • input（） 先打印括号内内容，等用户输入内容再进行下一步
  • 引入模块
    • random模块 文件开头导入模块import random，接着secret=random.randint(1,10)，引入1-10的随机整数。猜数字游戏代码块为下
import random
secret = random.randint(1,10)
print("*****i love fishc.com*****")
temp = input("猜个数字?")
guess = int(temp)
if guess == secret: 
         print ("猜对了")
else:
     if guess>secret:
               print("大大大了")
     else:
               print("小了小了")              
while guess!=secret:
     temp = input("不对哦，再试一次?")
     guess = int(temp)
     if guess == secret: 
         print ("猜对了")
     else:
          if guess>secret:
               print("大大大了")
          else:
               print("小了小了")
print("game over")
  • type() 查询数据类型；isinstance（a,b）a表示需要查询的变量，b填入数据类型：int/str/float等，符合则返回True，不符合返回False
  • 运算符：
          • / 正常操作
    • a+=1等效为a=a+1，-*/同理
    • //为整除，只保留整数位；%只保留余数
    • **为幂运算a**b为a的b次方，此操作符的优先级比左侧单目操作符高，比右侧单目操作符低：-3**2=9，3**-2=0.11111
    • 各操作符优先级：
    • 三元操作符a=x if 条件 else y条件为True则x赋值给a，为False则y赋值给a
    • assert 条件 条件为False时，程序自动崩溃并输出assertionerror错误
    • for 目标 in 表达式：循环体目标会遍历表达式中每个元素，表达式可以是字符串和列表。
    • range(start,stop,step)输出一个列表，从start开始到stop-1为止（不输入start默认从0开始到stop-1为止）step为步长即相邻两个数之间的差值。
    • break 退出循环体；continue跳出本次循环进入下一次循环
    • 列表名.append()在列表最后一位添加元素，列表名.extend([])在列表后插入另一列表内元素，列表名.insert(a,b)在列表位置a处插入b元素
    • 列表元素的删减
      • 列表名.remove(a)a为列表中某一元素，该指令将移除列表中第一次出现的该元素
      • del 列表名 (a)a表示元素位置，命令作用为删除该位置元素，不写则默认删除整个数组
      • 列表名.pop(a)a表示元素位置，命令作用为删除该位置元素并输出该元素，不写默认最后一位
    • 列表切片列表名[a:b]不改变原列表，输出元素从a处开始到b处前一位元素止的新列表，a不填默认从0处开始，b不填默认一直输出到原列表最后一位。
    • 列表名.count(a)统计列表中a出现的次数
    • 列表名.index(a,b,c)查询列表中a首次出现位置，b-c为查询范围，不填默认是整个列表
    • 列表名.reverse()列表倒序
    • 列表名.sort()将列表中的元素按一定规则排序，默认从小到大。
    • 采用extend方法拓展的列表类似于指针，所有指向此列表的变量均会改变。
